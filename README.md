
# Flappy Bird

Python Kullanılarak yapılmış Flappy Bird Clone.
## Kullanılan Teknolojiler

**Dil:** Python

**Modüller:** pygame , sys , random

  
## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/fkozdev/flappy-bird
```

Proje dizinine gidin

```bash
  cd flappy-bird
```

Kurulumları Yapın

```bash
  pip install pygame
```

Kodu çalıştırın

```bash
  python3 flappy.py
```
